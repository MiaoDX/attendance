>各个目录功能：

>> - 1. app

>>> - 1> controllers : 存放各个控制类

>>> - 2> models : 存放数据表输出model

>>> - 3> schemas: 存放相应的数据表schema

>>> - 4> views : 存放相关模板，以及前端页面

>>>> - a> pages: 所有页面

>>>> - b> public : 所有的公共依赖

>> - 2. config : 配置目录

>>> - routes.js : 路由配置文件

>> - 3. doc : 所有的文档

>> - 4. app.js : 系统入口

>> - 5. bin ： 运行目录
